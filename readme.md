# LessPass frontend

frontend application for [lesspass.com](https://lesspass.com)


 - vuejs
 - vue-router
 - vue-i18n
 - ava and xo for tests
 - webpack
 - ES6


## Tests

run frontend tests

    cd frontend
    npm install
    npm test


## Build

    npm run build

see [LessPass](https://github.com/lesspass/lesspass) project
