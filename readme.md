> LessPass Cordova App for [lesspass.com](https://lesspass.com)

## Build

    ./build.sh
    
## Run
    
    cordova run android
    
## Build Release

    cordova build --release -- --keystore=~/Android/android.jks --storePassword=... --alias=AndroidKey --password=...
