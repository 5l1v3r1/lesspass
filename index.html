Features

Manage your passwords directly from your browser


LessPass derives a site, a login and a master password to generate a unique password. You don't need to sync your password vault across every browser and device.

Memorize master password easily. Build a story in your head : "I like to eat in this luxury liner."

Use LessPass Database for complex passwords

We all know those sites that have special password rules. So you can't use the default options and have to remember the password is 6 characters long, only numbers. For this scenario we create what we call the "connected" version. Sign in, load or save complex password on LessPass Database.

To register an account on LessPass click on the icon in the LessPass component header.

Self Hosted

You can host your own LessPass Database. It's super easy. Just run a single command on your server.
